Chapter 1: Overview

With the release of Monster Hunter world and me finding a Monster Manual for Monster Hunter,
I wanted to create a Monster Hunter inspired Campaign.

This Document will contain an overview of the world and how you would build a character inside this world.

The only Rule is thats this is my homebrew world, so it will differ significantly from the established world of Monster Hunter, if you at any point say: "This isn't how this works in Monster Hunter",
you are subject to the possibility of me taking away a level.
